    [Description] Operating & retrieving Lipsync Mode setting state
  [Parameters]
    [Parameter] Manual
      [Description] PUT: turining its setting state to Manual / GET: indicating Manual setting status.
      [Note] About "Manual" setting of Lipsync Mode: When selecting "Manual" here, the audio delay value can be manually set.The value set by the "@MAIN:LIPSYNCTOTALDELAY" command will be the audio delay value.
    [Parameter] Auto
      [Description] PUT: turining its setting state to Auto / GET: indicating Auto setting status.
      [Note] About Lipsync Mode "Auto": The total audio delay value is controlled by the value set by "@MAIN:LIPSYNCTOTALDELAY" command.The total delay value contains the delay value read from the TV. (But if the delay value read from the TV is too big, it becomes the maximum value.)
