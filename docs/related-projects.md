# Related Projects

GitHub repositories and libraries for controlling Yamaha (and related)
receivers via serial or network protocols.

## Yamaha RS-232 (Legacy Protocol)

https://github.com/FireFrei/yrxv1500-mqtt
MQTT bridge for RX-V1500.

https://github.com/mrworf/yamahacontroller
RS-232 to REST API. Tested on Raspberry Pi with USB-to-serial adapter.

https://github.com/memphi2/homie-yamaha-rs232
ESP8266 Homie RS-232 control for Yamaha RX-Vxxx.

## Yamaha YNCA

https://github.com/mvdwetering/ynca
Python library for YNCA protocol. Supports 2010-2017+ models.
Includes interactive terminal for manual command debugging.

https://github.com/mvdwetering/yamaha_ynca
Home Assistant custom integration using YNCA (serial and network).
Most comprehensive list of supported YNCA models.

https://github.com/thomas-villagers/avsend
Sends YNCA commands via TCP. Tested with RX-V673, RX-A720, etc.

https://github.com/jtagg82/yncactl
Command-line YNCA control tool.

https://github.com/graememorgan/yamaha
Small Python YNCA script. Tested with RX-V675.

https://github.com/chewi/mpd-ynca
MPD to YNCA bridge.

## Yamaha YNCA Command Lists

https://github.com/akentner/yamaha-avr2mqtt
Contains YNCA command list text files for 2011 models
(RX-A810, A1010, A2010, A3010, RX-V671, V771) in docs/YNCA/text/.

https://github.com/neutmute/swimbait/files/490848/Yamaha-YNCA-Receivers.pdf
YNCA protocol specification PDF (hosted as GitHub issue attachment).

## Yamaha YNC (XML over HTTP)

https://github.com/KristopherKubicki/device-yamaha-rx
SmartThings integration. Supports RX-V2065, RX-V3900, RX-Z7, DSP-Z7
via YNC/XML protocol.

## Integra / Onkyo

https://github.com/miracle2k/onkyo-eiscp
Python library for eISCP (Ethernet ISCP) protocol.

https://github.com/mkulesh/onpc
Android app for Onkyo/Pioneer/Integra network control.

https://github.com/guikubivan/onkyoweb-php
PHP web interface for Onkyo receivers.

https://github.com/jhesch/integra
Integra serial control.

## Harman/Kardon

https://github.com/Teque5/avr_control
Harman/Kardon AVR serial control.

https://github.com/jherland/hifictl
Harman/Kardon serial control.
