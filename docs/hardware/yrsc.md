# Yamaha YRSC

The following models implement YRSC:

- RX-Z11
- RX-Z7
- RX-V3900

Although RX-V2065 implements YNC, on the RS-232 side it implements the
"legacy" Yamaha serial protocol (called `yamaha` in Seriamp) and not YRSC.
